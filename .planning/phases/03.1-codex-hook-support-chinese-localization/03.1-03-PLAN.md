## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| Hook payload text -> UI copy | Provider/category/message content is rendered in notification, tray, and detail text. |
| Unicode text -> Windows APIs | Chinese strings are passed to winotify, pystray, Typer, and MessageBoxW. |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-031-05 | Information Disclosure | provider/project display | accept | Provider and project name are intentionally shown to help the user identify which agent needs attention. |
| T-031-06 | Denial of Service | long messages in UI | mitigate | Preserve existing truncation behavior for notification/detail messages. |
| T-031-07 | Spoofing | provider display labels | accept | Provider label is derived from normalized internal provider value, not directly from untrusted raw payload when possible. |
</threat_model>
