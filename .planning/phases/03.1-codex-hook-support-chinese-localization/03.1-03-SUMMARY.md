# Phase 03.1 Plan 03: Chinese Notifications, Tray, and Detail Provider Display Summary

**One-liner:** Centralized Chinese-first display layer with provider-aware cooldown and notification eligibility, applied across notifications, tray menu, and event detail popup.

## Task Execution Summary

### Task 1: Add centralized display text helpers

**Commit:** `a2fda6c`

Created `src/notifier/core/text.py` with five helper functions:
- `provider_label(provider)` -- returns "Claude Code" or "Codex", degrades gracefully
- `category_label(category)` -- returns "需要授权", "等待输入", "任务已完成", "需要检查"
- `notification_title(event)` -- format: "{provider} - {project}"
- `notification_body(event)` -- format: "{cn_label} - {message}", truncates @ 200 chars
- `event_menu_label(event, relative_time)` -- format: "{provider} - {category} - {cn_label} ({time})"
- `relative_time_cn(timestamp)` -- returns "刚刚", "1 分钟前", "{N} 分钟前"

Added 28 tests asserting exact Unicode Chinese strings. All 47 tests passed.

### Task 2: Apply provider-aware Chinese notification copy

**Commit:** `5485a66`

Modified `src/notifier/core/notify.py`:
- `_build_body` delegates to `notification_body(event)` from text.py
- Added `should_notify(event)` -- suppresses ERROR and SessionStart events
- Cooldown key upgraded from `(project, category)` to `(provider, project, category)` to prevent cross-provider suppression
- Title uses `notification_title(event)` for provider-aware display
- Added message truncation to `notification_body` per T-031-06

Updated `test_notify.py` with new test classes: `TestShouldNotify` (6 tests), updated `TestBuildBody` (6 tests, Chinese assertions), updated `TestCheckCooldown` (7 tests, including provider-aware), updated `TestDispatchNotification` (8 tests). Added truncation test. 60 tests total pass.

### Task 3: Apply Chinese/provider labels to tray and detail surfaces

Modified tray and detail files:

- `app.py`: Tooltip "监控中" instead of "Monitoring", keeps "sessions"

Updated `test_menu.py` with Chinese assertions (10 tests), updated `test_tray_app.py` with "监控中" assertions (14 tests). Full suite: 192/192 pass.
